# AI Surveillance System

An AI-powered surveillance system that automatically detects and flags suspicious activity in video feeds using computer vision.

## Features
- Real-time video processing
- Suspicious activity detection
- Deep learning-based classification
- Automatic alerts

## Technologies Used
- Python
- OpenCV
- TensorFlow / PyTorch
- CNN (Convolutional Neural Networks)

## How to Run
```bash
python surveillance.py --video input_video.mp4
```

## Project Details
- **Computer Vision:** Uses OpenCV for video frame processing
- **Deep Learning:** CNN architecture for activity classification
- **Model:** Trained on surveillance footage datasets
